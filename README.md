## TUTORIAL INSTALL IN TERMUX

```bash
> pkg install git
> pkg install bash
> pkg install wget
> pkg install nodejs
> pkg install ffmpeg
> pkg install imagemagick
> pkg install mc
> git clone https://github.com/HannSykes/SykesBotWA
> cd SykesBotWA
> npm i
> npm start
```
